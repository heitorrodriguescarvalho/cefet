Variáveis

As Em JavaScript, as variáveis são utilizadas para armazenar dados que podem ser manipulados e utilizados ao longo do código. As variáveis podem ser declaradas usando var, let ou const, cada uma com suas próprias regras de escopo e hoisting. var possui escopo de função e sofre hoisting, enquanto let e const têm escopo de bloco, sendo que const é utilizada para declarar constantes cujo valor não pode ser reatribuído.

        
          var number = 123
          let string = "Hello, world!"
          const function = () => {}

Tipos de Dados

JavaScript possui uma variedade de tipos de dados que podem ser categorizados como primitivos e objetos. Os tipos primitivos incluem string (cadeias de caracteres), number (números), boolean (valores verdadeiro ou falso), null (ausência proposital de valor), undefined (valor não definido), symbol (valores únicos e imutáveis) e bigint (números inteiros arbitrariamente grandes). Objetos são coleções de propriedades, sendo fundamentais para estruturar dados complexos.

        
          const number = 123
          const string = "Hello, world!"
          const boolean = true
          const object = {}
          const array = []

Operadores

Operadores em JavaScript são utilizados para realizar operações sobre valores e variáveis. Operadores aritméticos (+, -, *, /, %) são usados para cálculos matemáticos. Operadores de comparação (==, ===, !=, !==, >, <, >=, <=) são usados para comparar valores. Operadores lógicos (&&, ||, !) ajudam a combinar ou inverter condições booleanas.Operadores de atribuição (=, +=, -=, *=, /=, %=) atribuem valores às variáveis.

        
          5 * 20 >= 80 //true

          !true //false
          
          const num = 3
          num *= 3
          num === 9 //true

Estruturas de Controle

Estruturas de controle em JavaScript permitem que o fluxo do programa seja alterado com base em condições específicas. Condicionais como if, else, else if e switch são usadas para executar diferentes blocos de código dependendo das condições. Laços como for, while e do...while permitem a execução repetitiva de um bloco de código enquanto uma condição for verdadeira. Comandos de controle de fluxo como break e continue podem interromper ou continuar a execução de laços conforme necessário.

        
          let num = 0
          for (let i = 0; i < 10; i++) {
            num += i
          }

          if (num > 10) {
            console.log(num)
          }

Funções

Funções são blocos de código reutilizáveis que realizam tarefas específicas e podem ser executadas quando chamadas. Em JavaScript, funções podem ser declaradas de várias formas, incluindo declarações de função (function), expressões de função e arrow functions (() => {}). Funções podem aceitar parâmetros e retornar valores, permitindo a modularidade e a reutilização do código. O escopo das variáveis dentro das funções é limitado ao corpo da função, proporcionando encapsulamento e evitando conflitos com variáveis globais.

        
          function print(text) {
            console.log(text)
          }

          let isOpen = false
          const handleClick = () => {
            isOpen = !isOpen
          }
